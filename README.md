# Inventory Management System

A full-stack inventory management application built using Java Spring Boot, MySQL, HTML, CSS, and JavaScript.

## Features

### Authentication
- User Registration
- User Login
- Logout functionality
- Session handling using localStorage

### Inventory Management
- Add Product
- View All Products
- Update Product
- Delete Product

### Additional Features
- Search products by name
- Sort products by price (ascending/descending)
- Low stock detection
- Highlight low stock products
- Dashboard statistics
    - Total Products
    - Low Stock Count
    - Top Category
- Export inventory data to CSV
